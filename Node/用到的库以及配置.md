## 1.nodemon 

使用原因：在node中修改文件之后还需要ctrl+c关闭，再重新输入node app.js重启，比较麻烦，所以用到的这个模块在你修改文件之后，自动执行上面两步指令，让开发爽一些。

```js
npm i -g nodemon 
安装到全局，方便。
// 使用方法
nodemon '文件名'
```
